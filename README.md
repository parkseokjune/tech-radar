# Tech Radar — 동적 딥테크 트렌드 + 예측 대시보드

딥테크를 좋아하는 사람들을 위한 트렌드 분석 시스템.
**고정 키워드 없이** arXiv 최근 논문에서 떠오르는 기술용어를 직접 발견하고,
각 용어의 시계열을 모아 **다음 분기를 예측**한다. 예측은 백테스트로 검증한다.

## 빠른 시작

```bash
python3 radar.py all       # 전체 파이프라인 (수집 → 발견 → 시계열 → 예측 → 대시보드)
open dashboard.html        # 결과 보기
```

브라우저는 인터넷 필요 (Chart.js CDN). `dashboard.html` 하나로 자체 완결.

## 명령어

| 명령 | 설명 | API |
|------|------|-----|
| `python3 radar.py harvest` | 최근/1년전 논문 수집 → 용어 발견 | ○ |
| `python3 radar.py discover [일수] [페이지]` | 용어 품질만 빠르게 점검 | ○ |
| `python3 radar.py extract` | 캐시 코퍼스로 용어 재추출 (오프라인 반복용) | ✗ |
| `python3 radar.py series` | 발견 용어들의 월간 시계열 수집 | ○ |
| `python3 radar.py analyze` | 지표·예측·백테스트 → data.json + 대시보드 | ✗ |
| `python3 radar.py build` | data.json으로 대시보드만 재생성 | ✗ |
| `python3 radar.py all` | 전체 | ○ |

`extract`/`analyze`/`build`는 캐시만 쓰므로 로직 튜닝 시 **API 없이 즉시 반복** 가능.

## 작동 원리

1. **동적 발견 (lift)** — 최근 75일 vs 1년 전 동기간 논문을 비교해, **지금 유독 많이 나오는** 다중어 기술구를 추출. (단일어 폐기, `of`/`and` 연결어 허용, `model`·`agent` 등 핵심명사 보존)
2. **시계열** — 발견 용어별 월간 논문 수 (2단어=정확한 구, 3단어+=토큰 AND 매칭).
3. **예측** — 감쇠 Holt 지수평활(α·β·φ 그리드서치)로 다음 3개월 예측.
4. **검증** — 최근 3개월을 숨기고 예측→실제 비교(MAPE). 부상 점수는 이 신뢰도로 가중.

## 시각화

- 🔮 **다음 분기 부상 예측** — 예측 성장률 상위, 실선(과거)+점선(예측)
- 🚀 **현재 모멘텀 랭킹** — 최근 3개월 vs 직전 3개월 + 스파크라인
- 📈 **모멘텀 × 볼륨 사분면** — 떠오름/뜨는 주류/성숙/니치 위치 (버블=논문량)
- 📊 **용어별 월간 트렌드** — 토글 가능한 다중 라인

## 한계 / 다음 단계

- arXiv 논문 수는 **연구 활동의 대리지표** — 산업 채택·펀딩과 다를 수 있음.
- 현재 코퍼스가 AI/ML 카테고리에 쏠림 (arXiv 논문량 자체가 그러함). → **도메인별 분리 추출**로 양자·바이오·소재 균형 개선 가능.
- 데이터 출처 확장: 펀딩(②), 특허, 채용공고, GitHub 스타 → 다중 신호 종합 모멘텀.
- 코퍼스 표본 확대 시 lift 안정성 향상 (현재 800편).
